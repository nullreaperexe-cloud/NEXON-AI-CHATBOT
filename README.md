# 🌟 Project Overview

**NEXON AI** is a next-generation AI chatbot designed with a premium cyberpunk terminal interface.

It features beautiful **glassmorphism**, animated **scanline overlays**, futuristic **neon green effects**, and an immersive hacker-style environment to deliver a unique AI experience.

The project runs entirely in the browser and communicates directly with **Google Gemini API**, requiring no backend server.

# 📥 Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/nullreaperexe-cloud/NEXON-AI-CHATBOT.git

cd NEXON-AI-CHATBOT
```

Or simply download the ZIP file from GitHub and extract it.

---

## 2️⃣ Generate a Gemini API Key

Visit **Google AI Studio**.

- Sign in with your Google account.
- Click **Get API Key**.
- Create a new API key.
- Copy your generated key.

---

## 3️⃣ Launch the Application

No installation or backend setup is required.

Simply open:

```text
index.html
```

in your preferred web browser.

> 💡 **Tip:** For the best development experience, use the **Live Server** extension in Visual Studio Code.

---

# 🚀 Getting Started

### Step 1 — Initialize Your Identity

Enter your preferred username when the terminal starts.

Click **Matrix Init** to continue.

---

### Step 2 — Connect Gemini

Paste your Google Gemini API key into the secure input field.

The chatbot will instantly connect to Google's AI service.

---

### Step 3 — Automatic Saving

NEXON AI automatically stores:

- Username
- API Key
- User Preferences

using your browser's **LocalStorage**.

---

### Step 4 — Settings

Click the ⚙️ gear icon in the bottom-left corner to:

- Update Username
- Change API Key
- Clear Local Storage
- Reset Settings

---

# 🛡 Privacy

Your API key is **never sent to any third-party server**.

NEXON AI communicates **directly with Google's Gemini API**, while all settings remain stored locally inside your browser.
